This repository contains a collection of practical Python programs . Each program demonstrates a specific concept or solves a common problem.

## List of Programs

2. **Sum of Two Numbers** - `sum_of_two_numbers.py`

4. **Greet the User** - `greet_user.py`
5. **Check Even or Odd** - `even_or_odd.py`
6. **Unique Elements in a List** - `unique_elements.py`
7. **Convert Celsius to Fahrenheit** - `celsius_to_fahrenheit.py`
8. **Area of a Circle** - `area_of_circle.py`
9. **Reverse a String** - `reverse_string.py`
10. **Check if a Number is Prime** - `check_prime.py`
11. **Factorial of a Number** - `factorial.py`
12. **Largest Item in a List** - `largest_in_list.py`
13. **Check if a Number is in a Range** - `number_in_range.py`
14. **Count Uppercase and Lowercase Letters** - `count_upper_lower.py`
